Q:磁斯达克是什么？

A:磁斯达克是中国领先的股权交易信息服务平台，致力于通过互联网的模式改造传统投资链条，推动中国企业家从实体经济投资向金融，互联网衍生品投资的转型。

投资人：

Q:投资流程是什么？

A:用户流程（投资人版）

Q:如何成为认证投资人？

A:登陆后点击“投资人”按钮，根据提示提交所需要的详细个人资料，完成后便可提交审核，审核通过后方可成为会员。

Q:投资人需要什么资质或者条件？

A:通过平台实名认证，且符合下列两个条件之一：

  1. （1）个人金融资产不低于300 万元或者最近三年个人年均收入不低于50 万元。
  2. （2）在净资产不低于1000万元的投资机构任职的专业风险投资人员且该机构。

Q:如何在磁斯达克平台中进行项目投资？

A:

  1. （1）注册成为磁斯达克用户。
  2. （2）申请投资人认证并通过审核。
  3. （3）点击查看项目详情，点击“我要投资”，磁斯达克会记录投资人的投资意向，并在未来的一到两天之内与投资人联系，确定投资意向。
  4. （4）磁斯达克发起项目方，投资人三方约谈，确定后签署投资合伙协议。
  5. （5）磁斯达克设立有限合伙公司，投资人打款至有限合伙企业，进行工商变更。
  6. （6）与项目方签署投资协议，磁斯达克将投资款支付给项目公司。

Q:上线如何了解现在项目的进展情况，在网站上能查到吗？

A:您可以在个人主页找到该项目查询项目的状态，项目状态有任何变化我们也会发送邮件和站内信通知。

创业者：

Q:融资流程是什么？

A:用户流程（项目方版）

Q:项目材料如何提交？

A:按要求填写项目信息，例如项目解决了用户什么需求、商业模式、未来发展规划、盈利模式及融资信息等，并提交商业计划书（上传的商业计划书仅向注册投资人公示），项目将由磁斯达克专业团队进行审核；我们将协助项目方进行项目资料优化，并辅助项目上线。

Q:在磁斯达克上融资的流程是什么？

A:

  • （1）项目方向磁斯达克平台发起融资申请，填写详细融资信息后并提交给磁斯达克。
  • （2）磁斯达克审核组会在一个工作日之内进行项目真实性初步审核，审核通过后交由专项组进行二次审核。
  • （3）由专项组和项目方接触，确定披露的信息及融资方案，包括融资目标金额、最低及最高融资金额等细节。
  • （4）细节确定后，项目方在线下载《磁斯达克平台服务协议》，打印转成纸质协议后填写详细信息，盖章并邮寄给磁斯达克。
  • （5）磁斯达克收到由项目方寄出的合同后，盖章并寄回给项目方。
  • （6）磁斯达克根据项目方提供的所有融资相关信息进行优化编辑，上线展示融资信息。
  • （7）开放项目方在线融资，投资人发起投资意向，磁斯达克发起项目方、投资人参与的三方约谈，确定后签署投资合伙协议，投资人支付投资款。
  • （8）办理交割手续，设立有限合伙公司及工商变更。
  • （9）变更完成后，平台将投资款支付给项目公司。

Q:需要满足什么条件才能在磁斯达克平台在线融资?

A:审核主要以考量项目方综合素质、阶段性工作执行效能，问题解决能力、以及创意点，行业痛点解决，项目复制难点，竞品分析，盈利模式可持续性等问题为主，需要审核人具备一定的问题分析能力、项目经验以及主观判断能力。

Q:什么样的项目更容易通过磁斯达克申请？

A:

  1. （1）磁斯达克平台投资人一般偏好互联网、科技、金融等方向的创业项目，这些类项目更容易通过申请。
  2. （2）对于传统行业的项目，如果是做传统产业转型互联网类的，也较容易通过申请。
  3. （3）拥有完善创业团队、清晰商业模式的创业项目更容易获得申请。
  4. （4）申请融资创业项目，项目信息填写的越完善，越容易通过合投申请。

Q:创业者来磁斯达克融资的优势

A:

  • （1）强大的产业助力

    在磁斯达克平台汇聚了一批在各个领域有影响力的产业投资人。他们能为项目提供强大的专业助力，有效提升创业成功率。

  • （2）高效标准化

    平台提供了项目商业计划书（BP）的标准化模版，你只需按要求填写相关内容，即可生成直观的商业计划书，向潜在意向投资人统一展示，还可以通过扫描二维码，在微信里便捷分享你的项目。

  • （3）资源面积广

    企业将会有多个股东，这样就可能有更多的投资人为企业提供资金及智慧资源，有效降低创业风险。
