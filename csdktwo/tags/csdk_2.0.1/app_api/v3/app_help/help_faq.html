什么是股权投融资？

股权投融资是指，公司出让一定比例的股份，面向普通投资者，投资者通过出资入股公司，获得未来收益。通过互联网渠道而进行融资的模式被称作股权投融资,或者称 之为“私募股权互联网化”。

股权投融资是不是涉及非法融资？

股权投融资不是非法集资。

根据《关于取缔非法金融机构和非法金融业务活动中有关问题的通知》规定，非法集资是指单位或者个人未依照法定程序经有关部门批准，以发行股票、债券、 彩票、投资基金证券或者其他债权凭证的方式向社会公众筹集资金。

股权投融资的流程是什么？

  • （1）融资方完善所有填写信息，磁斯达克筛选审核排期上线项目
  • （2）投资人完善信息缴纳保证金后通过磁斯达克审核，并开始在线选择项目
  • （3）上线项目开放给所有认证投资人浏览
  • （4）开放融资，投资人直接支付投资款至磁斯达克平台专户
  • （5）融资成功后开始办理交割手续，包括投资协议签署、设立有限合伙及工商变更；
  • （6）股权变更完成后，磁斯达克平台将募集款支付给融资方公司

投资人人需要符合的条件是什么？

  1. （1）具备成功退出经验的天使投资人或专业领投机构；
  2. （2）应充分认同平台的发展原则及方向；
  3. （3）在某个行业或领域有丰富的经验，独立的判断力，丰富的行业资源和影响力，很强的风险辨别能力；
  4. （4）具有一定的投资管理能力以及投后管理能力；
  5. （5）通过平台实名认证，且符合下列三个条件之一： a.个人金融资产超过100万元；b.年收入超过30万元；c.在投资机构任职的专业风险投资人员；

投资方是否可以直接参与融资公司经营？

除非有明确的合同约定，投资方一般不直接对项目拥有管理权，但可对创业项目提供必要的资源支持。

磁斯达克股权融资选择公司有什么标准，需要满足什么条件才能在磁斯达克平台上线融资？

审核主要以考量合伙人综合素质、阶段性工作执行效能，问题解决能力、以及创意点，行业痛点解决，项目复制难点，竞品分析，盈利模式可持续性等问题为主，需要审核人具备一定的问题分析能力、项目经验以及主观判断能力。
