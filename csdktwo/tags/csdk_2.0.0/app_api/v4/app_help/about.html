介绍

磁斯达克致力于通过互联网的模式改造传统投资链条，推动中国企业家从实体经济投资向金融、互联网衍生品投资的转型，结合金融全球化发展与信息技术创新，以健全的风险管控体系为基础，为广大企业与合格投资者提供专业、高效、安全的投融资平台。

磁斯达克2015年3月正式上线运营，结合全球金融发展与互联网+创新概念，在完整的风险管控体系基础上，为创业公司提供专业、可信赖的融资服务，为投资人提供跳过中介，降低成本一整套的资本投资、退出解决方案。融资涉及种子期至IPO的各个企业发展阶段。

  • 中文名称：磁斯达克
  • 英文名称：CISDAQ
  • 总部地点：北京市朝阳区
  • 上线时间：2015年

磁斯达克专业的募投管退服务：

  1. 1.募：汇集大量优质投资者形成最优质资本。
  2. 2.投：来自于各行业内精英，顶尖投资经理产业投资人和创业人群的优质项目。
  3. 3.管：每一笔投资均经过专业投资管理团队进行严格评审和筛选，所有程序和结果线上化对投资人直接呈现。
  4. 4.退：结合上市公司，优质券商和投资机构的多样化最优退出机制。

磁斯达克具备的核心价值：

我们直接拥有：

  • 1、优质的资本集合。
  • 2、丰富的项目渠道。
  • 3、专业的投资管理。

    • 我们对投资人直接提供？

  • 1、可靠的投资标。
  • 2、可视化的投资和投后管理全流程。
  • 3、以灵活多样化的最优退出机制最大化利益。
