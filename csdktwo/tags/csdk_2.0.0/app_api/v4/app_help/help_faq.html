1.什么是一级市场股权投资？

一级市场股权投资，是指投资于非上市公司股权，最终通过上市或者并购退出，获得高额收益的一种投资方式。

2.一级市场股权投资与炒股票的异同

一级市场股权投资与炒股票的相同点：


  • 都是对企业进行的一种股权投资，就是说都是出钱买公司的股票或股份;


  • 投资回报方式一样，都是作为企业小股东，获得被投企业利润分红或对自己所持股票（股份）溢价转让获得回报；


  • 投后干预程度一样：作为企业小股东，都一般不深度干预企业董事会的战略决策与总经理的日常管理。都只是以企业股东身份出席企业股东大会行使自己股东权利。

    • 一级市场股权投资与炒股票的不同点：


  • 投资企业阶段不一样 股权投资是对处于初创和发展阶段的企业进行投资，炒股票是对已经上市的企业进行投资；


  • 投资风险不一样 股权投资的核心在于看准项目创业者个人，项目创业者一般会欢迎投资人对其进行详细沟通并进行深度的背景调查；炒股票的投资风险涉及到证券市场制度体系、企业所处行业宏观发展环境、企业高管团队、企业发展战略与企业文化等许多复杂因素，上市公司一般会谢绝小股东与高管团队详细沟通与深度背景调查的要求；


  • 社会价值不一样 天使投资是支持和鼓励有创新精神、有创业激情的创业者（一般是年轻人），以成就创业者事业梦想来获得自己的投资回报；炒股票绝大多数股民只是单纯为获得投资回报为目的。
