三星电子改革企业文化向新创公司看齐

全球最大智能手机暨存储器芯片制造商三星电子(Samsung Electronics)周四宣布，计划采行类似新创公司的企业文化。三星高管将营造一个能够促进开放对话的工作环境，也将缩减管理层级，减少不必要的内部会议，简化报告流程，增加业务部门主管与员工之间线上讨论的频率。三星电子也将减少不必要的加班与周末工作，鼓励员工花时间陪伴家人或利用各种学习机会。全球员工超过30万人的三星电子一直面临改革其军事风格企业文化，以推动创新的压力。

松下下调营收目标

松下(Panasonic)将撤回到2018财年(截至2019年3月)合并销售额增至10万亿日元的业绩目标，调整为在不设期限的情况下，力争实现9万亿日元的目标。今后松下将把业务大致分成两个领域，把面向消费者的业务作为“BtoC”领域，力争使车载导航等汽车、住宅及家电(主要是白色家电)3个部门合计年销售额达到6万亿日元；面向企业的“BtoB”领域则将扩大高安全性相关产品等的业务规模，力争年销售额达到3万亿日元。

三井住友金融旗下公司将以负利率筹措资金

三井住友金融集团(SMFG)旗下的三井住友融资租赁公司24日表示将以负利率筹措资金，这将开日本国内民营企业之先河。三井住友融资租赁公司本月28日将发行被称为“商业票据”(Commercial Paper)的短期公司债，从市场筹措50亿日元资金，利息为负0.001%，6个月到期。即半年可获得2.5万日元利息。
