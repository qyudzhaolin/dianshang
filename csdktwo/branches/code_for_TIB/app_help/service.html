【磁斯达克平台】注册服务协议

【磁斯达克平台】注册服务协议为北京磁信信息技术有限公司和【磁斯达克平台】会员就【磁斯达克平台】众筹投资交易居间服务的相关事宜达成的协议。

一.定义

如无特别说明，下列用语在本协议中的含义为：

  • 1.【磁斯达克平台】/平台：指北京磁信信息技术有限公司或及合作机构提供的众筹投资交易居间服务系统；
  • 2.会员：指【磁斯达克平台】服务的客户；
  • 3.会员账户：指会员在【磁斯达克平台】有效开立的账户；
  • 4.平台服务：指平台借助因特网或其它技术为会员提供的众筹投资交易居间服务；
  • 5.交易指令：指会员向平台发出的与投资交易相关的申请、确认、指示等意思表示。

二.平台服务内容

1.服务内容

平台服务内容主要包括为项目发起人与支持者之间的交易提供平台网络空间、技术服务和支持、协助发布交易信息、提供交易管理服务、提供用户服务等, 您同意平台服务的具体内容由平台方根据实际情况进行调整。除非本协议另有其它明示规定, 平台所推出的新产品、新功能、新服务, 均受到本协议之规范。

平台非众筹项目的发起人，仅为平台上所进行的交易行为提供居间及其他相关服务。

2.服务费用

会员使用平台服务时，平台会向会员收取相关服务费用。各项服务费用详见会议使用平台服务时平台上所示之收费方式说明。平台保留单方面制定及调整服务费用的权利。

三.会员

1.平台会员须为中国籍自然人或纯内资中国企业，并具备如下条件：

a)如为自然人：

持有中国大陆（不包括香港特区、澳门特区及台湾地区）有效身份证件；

18周岁以上且能够独立承担民事责任；

个人持有的金融资产不低于300万元，或最近三年个人年均收入不低于50万元。

b)如为企业：

持有中国政府机关颁发的有效营业执照；净资产不低于1000万元。

2.仅满足如上条件且通过身份验证的会员方可获取平台上的众筹项目信息。一旦注册会员不再满足如上条件，应立即通知平台，并删除其从平台获取的一切众筹项目信息。

3.会员在注册时应上传最新的、真实有效的会员身份证件的彩色扫描件以进行身份验证，包括但不限于居民身份证或营业执照（副本）、法定代表人身份证明、组织机构代码证、开户许可证；并应提供最新的、真实有效的会员联系信息，包括但不限于常用地址、联系方式和紧急联系人信息等。

4.会员有义务维持并更新会员的资料，确保其为真实、最新、有效及完整。若个人会员提供任何错误、虚假、过时或不完整的资料，或者平台依其独立判断怀疑资料为错误、虚假、过时或不完整，平台有权暂停或终止会员账户，并拒绝会员使用平台服务的部份或全部功能。在此情况下，平台不承担任何责任，并且会员同意负担因此所产生的直接或间接的任何支出或损失。

5.如因会员未及时更新基本资料，导致平台服务无法提供或提供时发生任何错误，会员不得将此作为取消交易或拒绝付款的理由，平台亦不承担任何责任，所有后果应由会员承担。

6.会员不得使用或冒用他人信息注册为平台会员，否则平台将追究实际使用人的法律责任。

四.会员账户

  1. 1.开立会员账户是成为磁斯达克认证投资人的前提条件，开立会员账户后您可以享受到磁斯达克专业的在线股权投资服务。认证投资人可在线查看经过磁斯达克完整的审查体系调查通过、由磁斯达克专业项目组根据投资人的偏好精心筛选推荐的优质创业项目，并可完整享受由磁斯达克为投资人定制的线上、线下专业投资相关服务。
  2. 2.会员申请开立会员账户，需要缴纳十（10）万元人民币至临时账户作为会员诚意金。经财务审核确认后，临时账户即转变为正式的会员账户，会员通过会员账户按本协议的约定享受相关投资服务。一旦会员在磁斯达克平台成功完成一次投资行为，会员即可申请全额退还会员诚意金，并终身保留磁斯达克认证投资人身份。若自会员账户自开立之日起一年内未在平台成功发生过任何投资行为或申请取消磁斯达克认证投资人身份，平台有权从会员诚意金中扣除五（5）万元人民币作为当年的平台服务费；若在会员账户自开立之日起第二年内仍未在平台成功发生任何投资行为或申请取消磁斯达克认证投资人身份，平台同样有权从会员诚意金中扣除五（5）万元人民币作为当年的平台服务费。若会员账户金额为零，平台有权冻结磁斯达克认证投资人身份及其会员账户功能，直至充值后方可重新使用。

  3. 3.开通会员账户流程：

    身份验证——开通临时账户——缴纳会员诚意金——财务审核——开通会员账户。

    如申请人未能开通正式的会员账户，平台将向申请人退还其缴纳的全部会员诚意金。
