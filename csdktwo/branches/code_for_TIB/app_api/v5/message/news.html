轻松筹完成近 2000 万美元 B+ 轮融资

6月1日，社交众筹平台轻松筹向 36 氪确认完成近 2000 万美元B+ 轮融资，本轮融资由腾讯、IDG、德同资本、以及同道资本共同投资。轻松筹于 2014年8月12日 正式上线，是独立众筹工具平台众筹空间首个上线的子产品。从传播渠道上来说，轻松筹上的众筹项目主要是通过发起人自己的社交圈子进行传播，如微信朋友圈、社群等。即用户在轻松筹发起众筹之后，可以分享出去。

汽车之家第一季度净利润2.447亿

6月1日下午，汽车之家发布了截至2016年3月31日未经审计的2016年第一季度财报。财报显示，2016年第一季度，汽车之家净营收同比增长75.5%，至10.935亿元人民币(1.696亿美元)。调整后净利润在2016年第一季度同比增长49.1%，至2.944亿元人民币(4,570万美元)。

小米称买下微软1500项专利

2016年6月1日，小米(Xiaomi Corp.)宣布已从微软(Microsoft Co.)购买大约1,500项专利。小米购买知识产权是为了有一天能够在发展中市场以外的地区销售其设备，最终目标之一是在美国销售手机。小米还与微软签署了专利交叉许可协议，作为协议的一部分，小米将在该公司小米手机5、小米Max、红米Note 3和小米手机4S等设备上预装Microsoft Office和Skype。
